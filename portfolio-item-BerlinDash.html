Introducing AirBnB Infographic

Berlin AirBnB

  • Excel
  • Kaggle
  • Tableau
  • Qlik sense

For this project an infographic like dashboard was created in order to answer a series of questions made from extensive research of a dataset relating to Airbnb's./P>

A dataset was selected from kaggle, sorted and altered through Excel and questions were formed using the wants and needs of an Airbnb competitor. The question was then broken down and a rough paper prototype was made of how a dashboard could answer the given questions.

This design was then translated to Tableau and made interactive so that a change in one graph could effect another.

Upon completing the dashboard Infographic, i repeated the process on the software Qlik-Sense to create one that was near identical. The interface was easy to use but i believe tableau was more user friendly.
