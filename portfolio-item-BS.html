Introducing My Banking Software

Manage your accounts

  • Java
  • Netbeans

Created on Netbeans using Java, This Banking Software was one of my first large scale projects. In this banking system, users can enter their details to create either a current, savers or Isa account. There are two user types, user and Admin. Users are able to deposit and withdraw money from these accounts then are able to see their new balance. Admins can display and edit the customer list, branch address, and can find and display certain customers.

My Banking Software, also has the ability to print statements of the transactions made by the user with the click of a button.
